# CS5101_NLP_Group_Project

## Overview
This project focuses on news topic classification using Natural Language Processing (NLP) preprocessing techniques and Machine Learning models on the AG News Dataset.

The system classifies news articles into four categories:
- World
- Sports
- Business
- Sci/Tech

---

## Features
- Text preprocessing
- TF-IDF vectorization
- Machine Learning classification
- Model evaluation using accuracy, precision, recall, and F1-score
- Data visualization and confusion matrix analysis

---

## Technologies Used
- Python
- pandas
- scikit-learn
- nltk
- matplotlib
- seaborn

---

## Dataset
AG News Dataset:
https://www.kaggle.com/datasets/amananandrai/ag-news-classification-dataset
